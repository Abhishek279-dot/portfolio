## 🛠️ Technologies Used

- **HTML5** — Page structure & semantics
- **CSS3** — Styling, animations, responsive design
- **JavaScript** — Interactivity & dynamic content
- **Typed.js** — Typing animation in hero section
- **Font Awesome 6** — Icons throughout the site
- **Google Fonts (Poppins)** — Typography

---

## ✨ Features

- ✅ Fully **Responsive** — works on mobile, tablet & desktop
- ✅ Smooth **scroll navigation** with fixed navbar
- ✅ **Typing animation** (Data Scientist, FullStack Dev, AI-ML Engineer)
- ✅ **Hamburger menu** for mobile
- ✅ **Certificate modal popup** — view certificates on click
- ✅ **Timeline** for experience section
- ✅ **Hover effects** on cards, images & buttons
- ✅ **Resume download** button

## 🚀 Getting Started

To run this project locally:

```bash
# Step 1 - Clone the repo
git clone https://github.com/Abhishek279-dot/portfolio.git

# Step 2 - Open the folder
cd portfolio

# Step 3 - Open in browser
# Just open index.html in any browser!
```

No installations or dependencies needed — pure HTML/CSS/JS! 🎉
